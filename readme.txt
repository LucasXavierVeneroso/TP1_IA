Bem Vinde ao README do meu TP1 de Introduçao a IA

Para executar o programa, navegue no terminal ate a pasta em que se encontra o programa e depois apenas digite 'python TP1.py'
Nao se esqueça de ter o compilador de Python ja baixado no computador e atualizado. 
Depois de se digitar python TP1.py e apertar ENTER o programa ira esperar por entrada.
Coloque a entrada no formato especificado e apos apertar ENTER o programa te dara a saida.
